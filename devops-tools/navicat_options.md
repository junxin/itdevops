常用快捷键：

1. ctrl + q: 打开新查询窗口

2. ctrl + r: 运行当前窗口内的所有语句

3. ctrl + w: 关闭当前窗口

4. F6: 打开一个MySQL命令行窗口

---------------------------以下是然并卵的快捷键----------------------------------------

5. ctrl + n: 打开新查询窗口

6. ctrl + shit + r: 只运行选中的语句

7. ctrl + /: 注释

8. ctrl + shift + /: 取消注释

9. ctrl + l: 删除一行

10. F7: 运行从光标当前位置开始的一条完整sql语句

11. ctrl + d: 在表数据窗口上查看表定义
