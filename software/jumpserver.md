# jumpserver安装

- 安装依赖
```
yum -y install git python-pip gcc automake autoconf python-devel vim sshpass lrzsz
```
- 下载jumpserver

```
git clone https://github.com/jumpserver/jumpserver.git
```
> 注：不要安装在/root、/home 等目录下，以免权限问题

- 运行安装脚本

```
cd jumpserver/install/
./install.py
```
